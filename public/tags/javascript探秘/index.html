javascript作用域--执行环境

本文记录执行环境。

变量对象

变量对象是与执行上下文相关的数据作用域。它是与上下文关联的特殊对象，用于存储被定义在上下文中的变量和函数声明。

活动对象

当函数被激活的时候，活动对象就会被激活，这个活动对象就是该函数的变量对象。

作用域链

在当前函数中，可能会引用它的父函数变量，或者全局的变量，指向这些执行环境各自的变量对象引用组成一个作用域链。
